# Art Direction and Design

**Philosophy:** Andy Clarke's "Art Direction for the Web".

**Negative Constraints (Forbidden):**
-   **NO Symmetric Grids:** grid-cols-2 is banned for Hero sections.
-   **NO Generic Bootstrappy Look:** If it looks like a default template, it failed.
-   **NO Lorem Ipsum:** Use "Lekki Phase 1", "Victoria Island" context.

**Positive Constraints (Required):**
1.  **Clarke Grid System:** Use 2:3, 3:5, or Golden Ratio layouts.
2.  **Content-First:** Ask for the emotional goal (Trust vs Excitement) *before* coding layout.
3.  **Motion:** "Aura-level". Purposeful entry animations using Framer Motion.
